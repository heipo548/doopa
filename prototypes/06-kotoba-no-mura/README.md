# 06 — ことばの むら（体験版）

> おぼえた ことばで、せかいが かわる。…だれかが、あなたを かぞえている。

なぐらない村で目を覚ました、記憶のない主人公。村のどうぶつたちに **ことば** を教わるたび、
画面じゅうの `？？？` が「読める言葉」に変わり、**世界の色** までやわらかく変わっていく——
けれど 空の むこうで、だれかが あなたの **クリック・読み飛ばし・よそ見** を しずかに数えている。

戦わない。村では暴力は禁止で、みんな **ことばで いいあう**（言い争い）。
そして最後、ほこらの **かぞえうた** が、ゲームの中の あなたではなく、**画面の前の あなた** を言い当てる。

🎮 **遊ぶ**：このフォルダの `index.html` を開くだけ（依存ゼロ・外部画像/音源ゼロ・`file://` でも動く）

---

## この体験版で味わってほしいこと

1. **覚える言葉で世界が変わる**：言葉を覚えると `？？？` が読め、背景の色温度が 明↔暗 に傾く。**フィールドの名札も最初は ？？？ だらけ**——であった ひと・おぼえた ことばの分だけ、世界に "きみの語彙の注釈" がついていく
2. **戦わない言い争い**：相手の様子と言葉を読み、`やわらか`（ごめん・だいじょうぶ）か `とげ`（うるさい・だめ）かを選ぶ。やわらか＝世界が明るく／とげ＝早いが、世界が暗くなる
3. **メタの不気味**：ゲームは起動から ずっと、あなたの **総クリック数・読み飛ばし回数・よそ見（タブ非アクティブ）時間・無駄クリック** を裏で数えている
4. **審判（サンズ的）**：ラスト、神的存在「かぞえうた」が あなたの実挙動を **具体的な数字で** 言い当てる。だらけて遊ぶほど、言葉が皮肉に、不気味になる
5. **マルチエンド×消えない記憶**：選んだ言葉のトーン（明／中／暗）で結末が分岐。そして かぞえうたは **周回しても きみを おぼえている**（2周目は「また、きたね」、まえの なづけ まで言い当てる）

プレイ時間：**15〜25分** ／ 失敗してもゲームオーバーにならない（“すれ違い”として先へ進む）

## 遊び方

- **うごく**：マップをタップ／クリック・`WASD`・矢印キー
- **しらべる・はなす**：対象に近づいて クリック／`Space`・`Enter`
- **言い争い中**：`🔎 みみをすます／ようすをみる` で言葉を覚え、覚えた **言葉ボタン** を選ぶ
- `📖 ことば`：覚えた言葉の **図鑑**　`☰`：中断（セーブ／図鑑／タイトル）　`Esc`：ポーズ

---

## いちばんの肝：かぞえうた（審判）

村も もりも、表は ちいかわ的に やわらかい。でも 井戸の そこから、ずっと声がしている：

```
？？？… ？？？… ？？？…
```

`かず` を覚えると、それが はっきり読める：

```
いち、 に、 さん……
```

そして ほこらで `みてる` を覚えたとき、**数えていたのは “あなた”** だと分かる。
かぞえうたは、あなたが この体験版で 何回クリックし、何回 文を読み飛ばし、何回 よそを見たかを——**実際の数字で**読み上げる。

> ドパガキ（刺激を求めてスワイプし続ける脳）に、そっと鏡を向ける。これがこのプロトの背骨。

---

## 進行（全体フロー）

1. **ことばの むら**：もこ→`こえ`、とん→`なまえ`、ぴこ（はなが ないと教えてくれない＝もりへのフック）、村のけんかを仲裁→`ごめん`/`ありがとう`（とげ語 `うるさい`/`だめ` も拾える）
2. **もや の もり**：`みず`/`はな`/`き`/`そら` を覚え、けわしい顔の **もりの ぬし** と言い争い→`だいじょうぶ`/`まもる`/`みち`
3. **村に もどる**：ぴこに はなを あげて `すき`、なかなおりした ふたりから `いっしょ`。**うた** が現れ `かず` を教わる（井戸の声が読める＝不気味の幕開け／きたのほこらが解錠）
4. **かぞえうた の ほこら**：無音のなか かぞえうたと対面→`みてる` を覚える→**審判**→自分の `なまえ` を選ぶ（**ともだち／かぞえうた／沈黙**）→**エンディング**

覚えられる言葉：**全20語**。`ありがとう`/`いっしょ` は **あたたかい周**だけ、`あっち`/`きらい` は **とげの周**だけ＝20/20 には2周以上が必要。図鑑の未習得マスには **ありかのヒント** が出る。
さらに探索で覚えた `みず`/`はな` は、言い争いで **"理解を しめす ことば"** として使える（収集が道具になる）。

---

## 言い争いシステム（戦闘の代わり）

相手を“倒す”のではなく、**言葉のトーンで むきあう**。内部に2つの隠し値を持つが、**数値は画面に出さない**——様子を文章で伝える。

| 値 | 意味 | 動き |
|---|---|---|
| つうじ | 相手にどれだけ伝わったか | やわらか語で上がる。高い状態で“通じることば”を言うと **あたたかい解決** |
| わだかまり | 相手がどれだけ身構えているか | とげ語で削れる。0 にすると **とげの解決**（早いが、何も通じない） |

**動的難易度**：あなたの監視スコア（読み飛ばし・よそ見・連打が多いほど高い）が上がると、
“通じることば”に必要な つうじ が少しだけ増える＝だらけて遊ぶほど、世界がほんの少し手強くなる。

## アーキテクチャ（house 様式）

ビルドなし・依存ゼロ・外部画像/音源ゼロ。`<script>` を順に読み、各モジュールは `window` へ明示エクスポート。
フィールドは **canvas の見下ろしマップ**、テキスト/UI は DOM、音は **WebAudio 合成（8bit）**。

```
index.html        画面の器（シーン/HUD/オーバーレイ/スクリプト順）
style.css         パステル絵本テーマ＋トーン可変(明→暗)＋「？？？↔言葉」書き換え演出
js/
  data.js         言葉20語（図鑑ヒント付）・3エリア・NPC4体・言い争い・審判/なづけ/周回記憶の文章
  meta.js         ★本作の心臓：クリック/読み飛ばし/よそ見/無駄クリック/歩数の裏計測＋動的難易度＋審判生成
                  ＋ soul（かぞえうたの“消えない記憶”＝周回数・結末・まえの なづけ。newGame でも消えない）
  audio.js        WebAudio 8bit BGM（メロ+ベース+アルペジオ+ハットの2曲・トーンで沈む）＋全行動SE＋ほこら無音
  words.js        言葉システム（？？？↔習得語の描き換え・トーン・図鑑）
  state.js        状態機械・フラグ・持ち物・世界トーン(明暗)・進行ヒント
  save.js         localStorage セーブ（キー kotobamura_save_v1・軽い難読化・上書きのみ／削除しない）
  field.js        canvas 見下ろしマップの移動/衝突/近接
  argue.js        言い争いの状態機械（つうじ/わだかまり・やわらか↔とげ・warm/harsh）
  ui.js           描画切替・canvas描画・タイプライタ・図鑑・トーン反映・習得ポップ・審判演出
  main.js         入力配線・フィールドループ・遷移指揮・★メタ計測フック（全クリック/読み飛ばし/よそ見）
tools/
  ui-syntax-check.js    構文＋全画面render＋グリッド整合＋言い争い(warm/harsh)＋審判生成を JXA でヘッドレス検証
  playthrough-check.js  村→けんか→もり→ぬし→うた→ほこら→審判→なづけ→エンド を“操作経路”で通す結合テスト
docs/GDD.md       設計メモ（バイブル対応表つき）
```

## アクセシビリティ / 互換

- `prefers-reduced-motion` 尊重（書き換え/タイプ/演出を即時化）
- キーボードのみ／クリックのみ どちらでも進行可能
- 日本語フォント（Yusei Magic）は **読み込めなくても system フォントへ自動フォールバック**
- メタ計測は **ローカル完結**。外部への送信は一切なし（数えるのは“この場の あなた”）
- 周回記憶（soul）も localStorage のみ。きえないのは、かぞえうたの記憶だけ

## 実装しないもの（体験版の割り切り）

通常戦闘・レベル・装備・売買・オンライン。**「覚える言葉で世界が変わる」×「あなたが数えられている不気味」** に集中する。
