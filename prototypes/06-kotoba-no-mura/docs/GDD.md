# GDD — ことばの むら（DOOPA proto #06）

体験版の設計メモ。元になった企画バイブル（`ゲーム企画チェックシート.csv` の「あなたの記入欄」）への対応を明記する。

## 1. ログライン

> 記憶をなくした主人公が、なぐらない村で **ことば** を覚えていく。覚えた言葉で世界の色が変わるが、
> 空のむこうの神的存在「かぞえうた」が、画面の前の **あなたの実挙動**（クリック・読み飛ばし・よそ見）を
> しずかに数えていて、最後に それを言い当てる。戦わない・ことばで いいあう探索RPG。

## 2. バイブルの2本柱（このプロトが正面から作るもの）

1. **覚える言葉で世界の見え方が変わる**（CSV: コンセプト核／差別化／世界観）
   → `？？？` の描き換え（words.js）＋ 世界トーン（明↔暗）の変化（state.tone）。
2. **メタ：プレイヤーの操作ログを裏で数え、最後にサンズ的に皮肉る＝不気味**（CSV: 面白さの核／フック／ビジョン／スコープ）
   → meta.js が clicks/skips/blur/voids/fast を計測し、metaVerdict() が **実数字で** 審判を生成。神的存在が監視＝動的難易度（metaForgiveness）。

シリーズ内の棲み分け：02=口喧嘩RPG、03/04=言葉カードバトル＋神様が操作の癖を見る、05=戦わない探索で？？？が読める。
→ **06 は柱②（メタ監視＝かぞえうた）を主役に**しつつ柱①（世界が傾く）を能動化。03/04 が軽く触れただけの部分を中心に据える。

## 4. コアループ

`歩く → どうぶつに話す/調べる → ことばをもらう（？？？が読める・世界が少し色づく） → 言い争いで言葉を選ぶ（トーンが世界を傾ける） → 次のエリアへ` 
… を裏で **かぞえうた** が数え続け、ラストで一気に回収（審判）。

## 5. メタ計測の内訳（meta.js）

- clicks（総クリック＝ぽち）／ skips（タイプ中に送った＝読み飛ばし）／ reads（読みきってから送った）
- voids（空き地クリック＝手持ち無沙汰）／ fast（250ms以内の連打）
- blurCount・blurMs（タブ非アクティブ＝よそ見）／ idleMaxMs（最長無操作）
- togeSaid・yawaSaid（言い争いの選択傾向）

watch（0..100）= skips・voids・fast・blur・toge の重みづけ − reads の相殺。
- watch 高 → 審判が sharp（皮肉/不気味）＋言い争いが ほんの少し手強い（forgiveness↓）
- watch 低 → 審判が soft（やさしい）

審判は **やっていないことは言わない**（条件を満たした癖だけ並べる）＝不気味なほど具体的に。
計測は **ローカル完結・送信なし**。プレイごとに metaReset で数えなおす（＝“この場の あなた”を見る）。

## 6. エリアと言葉（20語）

- むら（village）：こえ・なまえ・ごめん・ありがとう（＋とげ：うるさい・だめ・あっち）
- もり（forest）：みず・はな・き・そら・だいじょうぶ・まもる・みち（＋すき）
- ほこら（shrine）：かず・みてる・ともだち（なづけ）／きらい は dark 寄りの伏せ札

トーン: やわらか語=+ / とげ語=− で習得時に微変化。言い争いの選択で大きく動く（やわらか+4 / とげ-7）。

## 7. 失敗の扱い

ゲームオーバーなし。言い争いは warm でも harsh でも“決着”し、進行フラグ（quarrel_done/mori_done）は立つ。
harsh は早いが世界が暗くなり、かぞえうたが それも数えている（togeSaid）。

## 8. v2（品質底上げ）で入ったもの

ゲームデザインrubric（Core Experience / Motivation / Feedback / Clarity / Agency / Polish）で自己採点し、
スコアを最も押し下げていた3要因（①入手不能語3つ＝図鑑が完成しない ②世界が読めない体験がフィールドに無い ③周回の記憶が無い）を順に潰した。

- **フィールド名札の ？？？化**：であった ひと・おぼえた ことばの分だけ世界に注釈がつく（tagWord）。
- **全20語が入手可能に**：ぴこ（はな→すき）／なかなおりの ふたり（いっしょ）／ぬし harsh（きらい）。
  warm周限定＝ありがとう・いっしょ／harsh周限定＝あっち・きらい → 20/20 は2周以上（周回設計）。
- **soul＝かぞえうたの消えない記憶**：周回数・結末・まえの なづけ。2周目の審判・タイトルの ささやきが変わる。
- **収集語の道具化**：みず/はな は言い争いで“理解を しめす ことば”（reactions）。
- **審判の拡充**：滞在時間・歩数・周回数も言い当てる。なづけに「かぞえうた」を名乗る3択目。
- **見た目/音**：multiply夜化・花びら/ほこりパーティクル・水のうねり・窓明かり・まばたき・世界パルス・
  2チャンネルBGM・タイトル浮遊？？？・図鑑ヒント・エンド統計（ことば n/20・むすび n/3）。
