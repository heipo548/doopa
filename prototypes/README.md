## ステータスの意味

- 🌱 **seed** — アイデアだけ・モックアップ段階
- 🟢 **active** — 動いてる・改善中
- 🧊 **frozen** — 一旦止まってるが概念は生きてる
- 🪦 **archived** — 試した結果クローズ、学びだけ残す

削除はしない（学びとして残す）。やめるなら `archived` にする。

## 新しいプロトを追加するには

[../CONTRIBUTING.md](../CONTRIBUTING.md) を読んでください。
ざっくり言うと:

1. `prototypes/XX-your-slug/` フォルダを作る（XX は連番）
2. 中に `index.html` と `README.md` 最低 2 枚
3. ルート `index.html`（ハブ）にカード追加
4. 上記の表に行を追加
5. PR

## ガードレール（プロジェクト全体）

新プロトのアイデアを出すとき、唯一の縛り:

- 🎯 ターゲットは **ドパガキ**（TikTok / Shorts で常に刺激を求めてる若者）
- それ以外は何でもアリ。失敗・未完成・しょぼいで OK、**出した数が偉い**

メカニクスの細かいガードレール（「3〜10秒で完結」など）は**プロト単位**で各自設定すること。
プロジェクト全体に押し付けない。
