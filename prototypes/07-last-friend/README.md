# 07 — last-friend

**さいごのともだち（体験版）**

## 一言コンセプト
ことばを覚えるたび、世界の ███ が「読める」に変わるRPG

## このプロト固有のガードレール
- 数値を見せない（ゲージ・こころ・神様の観察はすべて演出で語る）
- 30語・NPC8体・口げんか1戦の上限を守る（DEMO_SPEC §12）
- 外部通信ゼロ（フォント含め全ローカル、保存は localStorage のみ）
- 全テクスチャ実行時生成・全SE WebAudio合成（外部素材ゼロ）

## 起動
```bash
open https://heipo548.github.io/doopa/prototypes/07-last-friend/
# or ローカルで
cd prototypes/07-last-friend/app && npm install && npm run dev
```

## あそびかた（20〜30分）
おかで めをさます → はなを しらべる → むらで みんなに あいさつ →
くちげんかを ことばで とめる → きつねのこに あう → ほしの よる → あさ。
覚えたことばは 会話の返事・口げんか・看板の解読、ぜんぶに使う。

## 何を試したかったか
- 「未知の言葉＝███」が会話・看板・バトルの**全UIを貫く単一機構**として成立するか
- 語彙の増加だけで「世界が読めるようになる快感」を作れるか（数値レスで）
- 行動ログ（しらべた回数・覚えた語数）を最後に1行だけ突き返すメタ演出の不気味さ
- SQUARE ENIX GAME CONTEST 2026 応募作のコア機構の体験版検証（GAME_DESIGN.md / DEMO_SPEC.md 準拠）

## 技術メモ
- TypeScript (strict) + Phaser 3.90 + Vite。ソースは [app/](./app/)、このフォルダ直下がビルド成果物
- フォント DotGothic16（SIL OFL）をサブセット化して同梱。CDN・外部画像・音声ファイル不使用
- 全データ駆動: ことば30語 / 会話185ノード / マップ / バトルはすべて `app/src/data/` の純データ
- 起動時にトークン・参照の全検証（誤字があると起動しない）
- 自動テスト: `app/scripts/e2e-full.mjs`（新規開始→30語コンプ→クリアまでの完全自然プレイスルー）
- 制作: Claude Code（実装・セリフ・ドット絵・SE合成すべてコード生成）
