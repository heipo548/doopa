# SPEC.md — 体験版仕様の要約（DEMO_SPEC.md v1.0 準拠）

> 原本はユーザー手元の `GAME_DESIGN.md`（世界観・全体構想）と `DEMO_SPEC.md`（体験版仕様）。
> このファイルは実装が守るべき要点の写し。仕様変更はここではなく原本側で行う。

## プロダクト
- さいごのともだち（体験版）/ TypeScript strict + Phaser 3.80+ + Vite / Node 20+
- 内部解像度 320×180・pixelArt・整数スケール・60fps / PC ブラウザのみ
- 永続化は localStorage のみ。**外部通信（fetch/XHR/WS）は一切禁止**
- フォント DotGothic16 woff2 同梱（CDN禁止）/ 日本語のみ
- `npm run dev` / `npm run build` → dist/ が静的サーバで動く

## 操作
矢印・WASD=移動 / Z・Enter・Space=決定 / X・Esc=キャンセル / C=ことばノート /
M=ミュート / F1=デバッグ（`?debug=1` のときのみ・通常UIから到達不可能）/ マウスは補助

## コアシステム
- **Word**: id/text/category(kind|sad|dark|world|truth)/hint/prereq/effects。30語固定（world14・kind11・sad4・dark1・truth0）
- **learnWord(id, source)**: 未習得時のみ習得演出（中央に語が浮く・コロンSE・世界の彩度up・スキップ可）
- **テキストマスク**: 全テキストは `{token}` 形式。未習得→ █ノイズ（0.5秒ごと1文字ゆれ）、習得済→ 語。描画時に毎回解決するので過去の看板も自動で読める。起動時に全データを走査し未定義トークンが1つでもあれば**起動エラー**
- 1文に未知語トークンは最大2つ
- **会話エンジン**: DialogueNode（speaker/text/lie/if/choices/palette/actions/next）。顔アイコン24×24・タイプライタ30ms/字・Zで全表示→次へ
- **ことばパレット**: 覚えた語から選んで返す返答UI。accept[語]→次ノード、default必須（会話を破綻させない）
- **バトル（口げんか）**: ターン制・なかなおりゲージ（gaugeStart50/max100・数値非表示）。matrix[語]→ゲージ変化、categoryMatrix が既定値。ゲージ0=敗北ではなく retryNode→30から再開。完全データ駆動（本編15戦の母体）
- **WordEffect**: reveal_sad（かなしい習得→きつねのこに涙差分）/ lie_shake（lie:true文が±1pxふるえ）/ unlock_choice は Cond.hasWord で実現
- **テレメトリ**: playMs/sessions/keyPresses/clicks/steps/examines/dialogueAdvances/wordsLearned/battlesWon/idleMs。key=`lastfriend.telemetry.v1`、5秒ごと+visibilitychange保存。**ゲーム中は一切表示しない**（使うのは E6 の実値と F1 のみ）。外部送信は永久に禁止
- **セーブ**: オートのみ。key=`lastfriend.save.v1`。契機=各イベントE0〜E7完了+visibilitychange。localStorage 消去で完全初期化
- **タイトル状態機械**: 初回=はじめから/？？？(選べない・カーソルでノイズ) → セーブあり=つづきから追加 → クリア後=？？？が押せて「（まだ ことばが たりない）」1行→戻る

## アート/音
- 全テクスチャは BootScene で Graphics から実行時生成（外部画像ゼロ）
- 16色固定パレット（gfx/palette.ts）以外の色は使わない
- 主人公=白い丸+黒目だけ。NPC8体は体色+耳/しっぽで差別化
- SE は WebAudio 合成のみ（決定/文字送り/習得アルペジオ/ゲージ/ノイズ0.1s）。BGMは任意（昼+夜アレンジ）

## イベント E0〜E7
- E0 おか: 読めないOP（ほぼ███）→ 花で「はな」（**開始3分以内**）→ みはらしで「そら」「かぜ」
- E1 うさぎ: ほぼ███のセリフ→「こんにちは」習得→**初パレット**で返す成功体験→村へ、Q1（4人にあいさつ）開始
- E2 自由探索: けいじばん「むら」/ カエル「パン」「おいしい」Q2（おばあにパン→「ありがとう」）/ モグラ「はたけ」/ はたけしらべ「つち」/ ことり「すき」/ おばあ「だいじょうぶ」「むかし」「おもいで」/ 水辺「みず」/ 木「き」/ とびら「いえ」
- E3 口げんか: りす×ふくろう（どんぐり）。直前にうさぎが「ごめんね」を教える。3フェーズ（いかり→かなしみ→いじ）。勝利で「なかなおり」「いっしょ」「どんぐり」
- E4 きつねのこ: lie:true の強がり→ 選択【ちがうと いう】=「うそ」習得(dark+1)・逃走→木の下で「さみしい」「かなしい」(reveal_sad) /【だまって すわる】=うそ未習得(kind+1)。合流: ごめんね→なかなおり→「きみは ぼくの なに？」→ すき+いっしょ 習得済なら**「ともだち」が合成され**、本人に言える（クライマックス）→ ゆうがたへ
- E5 うた: ほしみだいに全員集合。「うた」「ほし」（→よる）「よる」「たのしい」
- E6 かみさまの1行: 就寝後・真っ暗・**初めてマスクなしで全部読める**・無音・1行ごとに長い間。`きみは ことばを {n}こ おぼえた / きみは ものを {m}かい しらべた / (dark>=1のみ) きみは うそを しった / また みている ね / ――では また`。n,m はテレメトリ実値
