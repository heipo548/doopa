# GDD — 名前を知らない世界（体験版）

DOOPA プロト#05 の設計メモ。仕様書（依頼文）を実装に落とすときの判断を記録する。

## 1. コンセプト

言葉を知らない主人公が、旅の中で少しずつ言葉を覚え、世界を理解できるようになっていく探索RPG。
**成長 = 敵を倒して強くなることではなく、言葉を得て世界の意味が増えること。**

ターゲットはドパガキ。短い刺激ではなく「**覚えるたびに景色の意味が変わる**」という、
スワイプでは得られない“理解の快感”を 15〜25分で体験させる。

## 2. 中心メカニクス：言葉の描き換え

すべての読めるテキストは `{wordId}` を含むテンプレで書く。`words.renderText()` が
現在の習得状況（0 未習得 / 1 断片 / 2 習得）で**毎回描き換える**：

| 状態 | 表示 | 見た目 |
|---|---|---|
| 未習得 | `？？？` | にじみ（blur）・くすんだ紫 |
| 断片 | `ま…て…` 等 | 点線アンダーライン |
| 習得 | `待って` 等 | マーカー強調＋覚えた瞬間に金色ポップ（`.is-reveal`）|

→ 新しい語を覚えると、**過去に見た文まで遡って読める**。再描画だけで成立するのが強み。
名札（オブジェクト頭上）・会話・遭遇のセリフ・調べ文、すべて同じ仕組み。

## 3. 基本ループ（3回繰り返す）

探索 → 分からないものに出会う → 観察/耳をすます/対話 → 言葉の断片を得る →
覚える → 表示が変わる → 選択肢が増える → すれ違いが解ける → 先へ。

実装：森（水・花）→ 橋（遭遇A）→ 村（会話＋遭遇B＋ラスト）。

## 4. 遭遇システム（戦闘なし）

3 つの内部値 `u 理解度 / w 警戒度 / t 緊張度`。**数値は非表示**、`moods` のしきい値で
“相手の様子”を文章化（読解体験を優先）。行動 → 効果加算 → 文 → 学習/セリフ更新 → 核のことば接近 → 決着。

- 行動データは `data.ENCOUNTERS[id].actions`。`ladder`（耳をすますの段階学習）、`enableWhen`
  （`knows/minU/has` 等の条件で選択肢の出現）、`route:'success'|'fail'` で決着を記述。
- 失敗してもゲームオーバーにしない＝“すれ違い”。最低限の語は身につけ、フラグで先へ。
- 橋：核 `危ない`。耳をすます×3 で `待って→危ない→渡る`、橋を見るで `橋`、
  危ないを理解して「橋から離れる」で **成功（ことばが届いた）**。進みすぎ（緊張限界）で **失敗（落ちる）**。
- 子ども：核 `探す`。見るで `泣く`、耳をすますで `お母さん→探す`、理解が進むと自動で `こわい`、
  花を見せると警戒↓。「一緒に探す」/「村人を呼ぶ」で成功。近づきすぎ（警戒限界）で失敗。

## 5. フィールド（canvas 見下ろし）

- タイル地形（`.`草 `=`道 `B`橋板 `T`木 `~`川）＋オブジェクト（look/npc/encounter/exit/scenic）。
- 移動：クリック目標へ補間 or 矢印/WASD。タイル当たり判定で壁ずり。
- **もや**：`vocabCount` が増えるほど晴れる放射グラデ（読めなさの可視化）。
- 名札：オブジェクト頭上に `？？？`／語。覚えると景色が読めるのを“地図上でも”感じる。
- 橋の遭遇は **近づくと auto で開始**（板に踏み込む緊張感）。doneFlag 後は ただの風景。

## 6. 失敗設計

敗北ではなく“すれ違い”。相手が去る／追加の語が得られない／後で別の形で分かる。
橋失敗＝川に落ちて村人に助けられる（`危ない`だけは身体で覚える）。
子ども失敗＝逃げられるが、母から事情が分かる。

## 7. スコープ（実装しない）

通常戦闘・レベル・装備・売買・デッキ構築・複数エンディング・長大クエスト・ステータス育成。
→ 「言葉で世界が読めるようになる快感」一点に集中。

## 8. 拡張余地（本編に向けて）

言葉デッキ（持ち込む語を選ぶ）／文法システム（語を組み合わせて文を作る）/
誤解システム（一部だけ知る語で意味を取り違える）／抽象語（約束・嘘・さよなら…）。

## 9. 技術メモ

- ビルドなし・依存ゼロ・外部画像ゼロ・`file://` 動作。`<script>` 順読み＋`window` 明示エクスポート。
- セーブは `namae_save_v1`、**上書きのみ**（削除APIを使わない＝プロジェクト安全方針）。
- `prefers-reduced-motion` 尊重。フォントは Yusei Magic→system フォールバック。
- 検証：`tools/ui-syntax-check.js`（構文＋全画面render＋遭遇論理）と
  `tools/playthrough-check.js`（操作経路の通しプレイ）を JXA でヘッドレス実行。
