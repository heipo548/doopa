# 05 — 名前を知らない世界（体験版）

> ことばを 覚えるたびに、せかいに 意味が 増えていく。

言葉を知らない主人公が、**観察・耳をすます・対話**で少しずつ言葉を覚え、
画面じゅうの `？？？` が ひとつずつ「読める言葉」に変わっていく——**戦わない探索RPG**。

成長とは、敵を倒して強くなることではない。**言葉を得て、世界の意味が増える**ことである。

🎮 **遊ぶ**：このフォルダの `index.html` を開くだけ（依存ゼロ・`file://` でも動く）

---

## この体験版で味わってほしいこと

1. 最初は世界が分からない（多くが `？？？`）
2. 言葉を覚えると、**同じ景色の意味が変わる**
3. 相手の言葉が **徐々に読めるようになる**
4. 「戦う」のではなく「**理解する**」ことで道が開ける
5. 最後に「**ともだち**」を覚えて、物語的なカタルシス

プレイ時間：**15〜25分** ／ 失敗してもゲームオーバーにならない（“すれ違い”として先へ進む）

---

## 遊び方

- **移動**：マップをクリック / 矢印キー・WASD
- **しらべる・はなす**：対象に近づいて クリック / `Space`・`Enter`
- **遭遇中**：選択肢ボタンを選ぶ（特に **耳をすます** が基本行動）
- `☰`：中断メニュー（セーブ／覚えた言葉一覧）　`Esc`：ポーズ

---

## いちばんの勝負どころ：壊れた橋

向こう岸の村人が、けわしい顔で何かを叫んでいる——最初は **怒っているように見える**。

```
？？？！ その ？？？、？？？！ ？？？な！
```

**耳をすます**を重ねると、`待って` → `危ない` → `渡る` の順に音が言葉になり、橋を調べれば `橋` を覚える。
すると同じ叫びが、こう読めるようになる：

```
待って！ その 橋、危ない！ 渡るな！
```

怒っていたのではなく、**危険を知らせようとしていた**。
理解したうえで「橋から離れる」と——**ことばが届く**。戦闘は一切ない。

> 戦わないけど、ちゃんとゲームになってる。ここが伝わるかが、このゲームの肝。

---

## 進行（全体フロー）

1. **目覚めの森**：水辺で `水`、花で `花`（花を1本もらう）
2. **壊れた橋**：村人と遭遇。`待って`/`危ない`/`渡る`/`橋` を覚え、**理解して** 道がひらく
3. **小さな村**：老人と話し `人`、家で `村`、母と話し `子ども`
4. **泣いている子ども**（メイン遭遇）：`泣く`/`こわい`/`探す`/`お母さん`/`一緒`
5. 子を母の元へ → ラストで **`友達`** を覚える → 体験版エンド

覚えられる言葉：**16語**（水・花・人・村・橋・待って・危ない・渡る・子ども・泣く・こわい・探す・お母さん・一緒・ありがとう・友達）

---

## 遭遇システム（戦闘の代わり）

相手を“倒す”のではなく、**言葉と様子を読み取り、間合いと行動を選ぶ**。
内部に 3 つの値を持つが、**数値は画面に出さない**——「相手の様子」を**文章**で伝える。

| 値 | 意味 | 効果 |
|---|---|---|
| 理解度 | 相手の意図がどれだけ分かったか | 高いほど `？？？` が減り、成功選択肢が見えてくる |
| 警戒度 | 相手がどれだけ身構えているか | 高すぎると相手が逃げる／会話が打ち切られる |
| 緊張度 | 場面が破綻するまでの危険度 | 高すぎると失敗（橋なら “落ちる”） |

「核のことば」（橋＝`危ない` / 子ども＝`探す`）に近づくほど、道が開ける。

## アーキテクチャ（house 様式）

ビルドなし・依存ゼロ・外部画像ゼロ。`<script>` を順に読み、各モジュールは `window` へ明示エクスポート。
フィールドは **canvas の見下ろしマップ**、テキスト/UI は DOM。

```
index.html        画面の器（シーン/HUD/オーバーレイ/スクリプト順）
style.css         パステル絵本テーマ＋「？？？↔言葉」書き換え演出
js/
  data.js         言葉16語・3エリアのマップ/オブジェクト・NPC・遭遇・エンディング文
  audio.js        WebAudio 環境音とSE（合成音のみ）
  words.js        言葉システム（？？？↔習得語の描き換え・断片・習得）★本作の心臓
  state.js        状態機械・フラグ・持ち物・進行ヒント
  save.js         localStorage セーブ（キー namae_save_v1・上書きのみ／削除しない）
  field.js        canvas 見下ろしマップの移動/衝突/近接/もや
  encounter.js    遭遇の状態機械（理解/警戒/緊張・核のことば・成功/失敗）
  ui.js           描画切替・canvas描画・タイプライタ・習得ポップアップ
  main.js         入力配線・フィールドループ・遷移指揮（タイトル〜エンド）
tools/
  ui-syntax-check.js    構文＋全画面render＋遭遇の論理を JXA でヘッドレス検証
  playthrough-check.js  森→橋→村→子ども→母→ラスト→エンド を“操作経路”で通す結合テスト
docs/GDD.md       設計メモ
```

### 検証（node 不要・JXA）

```sh
# このフォルダで実行（cwd 依存）
osascript -l JavaScript tools/ui-syntax-check.js
osascript -l JavaScript tools/playthrough-check.js
```
