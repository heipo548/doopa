# 『勇者、退職させます。』体験版仕様書（2Dドット版 Ver.0.3）

## 1. 企画概要
- **タイトル**：勇者、退職させます。
- **ジャンル**：2Dドット・魔王軍運営アドベンチャー
- **プレイ時間**：1周 15〜25分
- **コンセプト**：魔王は勇者を倒せない。だから、勇者が戦う理由をなくす。

プレイヤーは魔王となり、魔王城の中を歩き回りながら部下に指示を出す。勇者を攻撃するのではなく、広報・人事・福祉・諜報などの施策を通じて、勇者パーティーの心境や世間の見方を変えていく。肝は「不自由の中で何でもやる」こと。

## 2〜5. ストーリー設定（要約）
- 先代魔王と王国は戦争していた。現魔王は和平を選び、人間を襲わない。
- 和平条約の証として魔王は **「和平の腕輪」** をつける。誓約：**魔王が人間を傷つけると、魔族の村を守る結界が消える** → だから勇者を攻撃できない。
- 魔王軍の支援活動（村の無料診療所・橋の修理・ダンジョン安全看板・孤児院開放・交易所）が、王国に **「侵略準備」** と誤解される → 新勇者レオンが任命される。
- 冒頭：玉座の間の定例会議（診療所の回復魔力／落とし穴の安全柵）の最中に、兵士が「勇者任命・魔王討伐」を報告。魔王「……なぜ？」「診療所だぞ？」

## 6. チュートリアル
プレイヤーに理解させる：①魔王は攻撃できない ②魔王軍は支援活動 ③王国に誤解されている ④戦うのではなく誤解と戦意を解く。
選択肢「攻撃する／村を焼く／呪いをかける／会議を開く」。上3つは腕輪が光りベルゼに止められる。最終的に「会議を開く」のみ進める。コア提示：**勇者を倒すのではなく、勇者が戦う理由をなくす。**

## 8〜9. ゲーム構造
全10ターン制（1ターン＝1週）。勇者は10週後に魔王城へ到達。それまでに討伐意欲をなくす／パーティー解散／評判で討伐命令を止める。
1ターンの流れ：①魔王城フェーズ（探索）②施策選択（1ターン1施策）③結果フェーズ（ドット演出）④勇者観察フェーズ（水晶玉で会話を見る＝本作の肝）⑤次ターンへ（接近）。

## 10〜11. 表示パラメーター（3つだけ・数値非表示）
状態名・アイコン・表情・ドット演出で見せる。
1. **勇者の迷い**：討伐一直線／違和感／迷い／葛藤／剣を置きそう → 最大で「勇者、退職」エンド
2. **仲間の不満**：団結／小さな不満／温度差／口論寸前／解散寸前 → 最大で「パーティー解散」エンド
3. **世間の評判**：極悪魔王軍／警戒／意外とまとも／好意的／支持拡大 → 最大で「共存条約」エンド

## 12. 非表示パラメーター（裏側）
勇者系（敵意・疲労・王国信頼・父親コンプレックス・魔王理解）／仲間系（戦士待遇不満・僧侶倫理迷い・魔法使い研究欲・結束）／世論系（村人支持・教会反発・王国広報リスク・誤解度）。会話・イベント・分岐・演出に変換する。

## 13. UI方針
数値を出さない。毎ターンの判断軸は3つ（勇者本人を揺らす／仲間から崩す／世間を味方に）。効果は会話とドットで見せる（「あ、効いてる」）。

## 14〜15. 魔王城マップ（1フロア）
玉座の間／広報室／人事室／福祉室／諜報室／水晶部屋／廊下／ニュース掲示板。
- 玉座の間：ベルゼ（軍師）。状況確認・ヒント・次ターンへ。
- 広報室：パズズ（明るい広報、炎上に弱い）。勇者の迷い・世間の評判。
- 人事室：リリム（採用・待遇比較が得意）。仲間の不満。
- 福祉室：オーク看護長（見た目こわいが優しい）。世間の評判・勇者の迷い。
- 諜報室：シャドウ（寡黙な忍者）。直接は上げにくいが、次以降の施策を強化・隠し会話解放。
- 水晶部屋：勇者パーティー観察。

## 16. 施策一覧（9）
1. 魔王軍ホワイト企業宣言（広報・世間）
2. 戦士向け転職広告（人事・仲間）
3. 村に魔王軍クリニック開設（福祉・世間/迷い、王国に前線基地と誤解される）
4. 勇者の過去を調査（諜報・下準備、父に認められたい が判明）
5. 魔王、公開謝罪会見（広報・迷い）
6. ダンジョン安全基準の導入（福祉・世間、勇者が進みやすくなる）
7. 王国の発表に反論する（広報・世間、誤解報道イベント後）
8. 僧侶に孤児院見学を招待（福祉・仲間/迷い、教会反発）
9. 魔法使いに研究資料を送る（人事・仲間）
※各施策に固有の「勇者側会話」がある（仕様書本文参照）。

## 17. イベント
1. 王国の誤解報道（評判が「意外とまとも」以上で発火、評判一段下げ・反論施策解放）
2. 和平の腕輪が光る（攻撃的選択時）
3. 勇者の夜の独白（迷いが「迷い」以上で発火、以降迷いが進みやすい）
4. 戦士、求人を隠し持つ（不満が「温度差」以上で発火、人事施策強化）

## 18. エンディング
- A 勇者、退職（迷い最大）：剣を置く→地元でパン屋開業、魔王軍から花輪
- B パーティー解散（不満最大）：戦士→警備部、僧侶→医療支援、魔法使い→研究所、勇者は城へ向かわない
- C 共存条約（評判最大）：王国が討伐命令撤回
- D 最終面談 成功（10ターン終了・裏フラグ十分）：会議室で面談、施策で説得選択肢が変わる
- E 討伐続行（面談失敗）：勇者が剣を抜く→腕輪が光る→「法務部を呼びます」コミカルなバッドエンド

## 20. 仕様の核
パラメーターを管理するゲームではなく、勇者たちの反応を観察するゲーム。見せる状態は3つだけ。裏で感情値・フラグを管理し、会話と演出に変換する。毎ターン考えるのは「勇者本人を揺らす／仲間から崩す／世間を味方につける」の3択。
