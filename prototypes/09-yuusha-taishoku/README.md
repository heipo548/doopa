# 09 — yuusha-taishoku

**勇者、退職させます。（体験版）**

## 一言コンセプト
魔王は勇者を倒せない。だから、勇者が戦う理由をなくす——和平の腕輪で攻撃できない魔王となり、広報・人事・福祉・諜報の施策で勇者パーティーの心と世間の見方を変えていく、2Dドットの魔王軍運営アドベンチャー。

## このプロト固有のガードレール
- **数値を見せない**。表に出す状態は3つだけ（勇者の迷い／仲間の不満／世間の評判）。状態名＋顔＋段階バー＋ドット演出で見せる。裏では細かいフラグを持つが、すべて会話と演出に変換する
- 魔王は人間を攻撃できない（攻撃すると魔族の村の結界が消える）。**殴る以外の手**でしか進めない、が背骨
- 毎ターン考えるのは3択だけ：**勇者本人を揺らす／仲間から崩す／世間を味方につける**
- 全ドット絵を実行時に Canvas 生成・全SE/BGM を WebAudio 合成（外部画像/音声ファイル不使用・ビルド不要）
- 体験版スコープ＝全10ターン・5エンディング（15〜25分）

## あそびかた
1. **歩く** — 魔王城（1フロア）を歩き、各部署のNPCに話しかける
2. **施策を1つ** — 広報/人事/福祉/諜報で、今週の手を1つだけ打つ（1ターン1施策）
3. **演出を見る** — 村に診療所が建つ、戦士が求人を拾う…短いドット演出
4. **水晶で観察** — 水晶部屋で勇者パーティーの会話・隊列・表情の変化を読む（＝本作の肝「あ、効いてる」）
5. **次の週へ** — 玉座のベルゼに話しかけて週を進める。勇者は10週後に城へ到着する

3つの可視状態のどれかが最大になれば即エンディング（退職／解散／共存）。10週を越えると最終面談（説得成功 or 討伐続行）。

## ファイル構成
素の HTML/JS（`<script>` 直読み・読み込み順は `index.html` 末尾の契約コメント参照）。
- `js/data.js` — 全コンテンツ（キャラ・マップ・9施策・4イベント・5エンディング・会話）
- `js/state.js` — ゲーム状態（可視3メーター＋隠しフラグ＋セーブ）
- `js/sprites.js` — ドット絵を実行時生成（顔・チビキャラ・家具・演出・エンディング絵）
- `js/audio.js` — WebAudio で効果音/BGMを合成
- `js/field.js` — 魔王城の探索（タイルマップ・移動・調べる）
- `js/crystal.js` — 勇者パーティー観察（隊列・姿勢・表情で状態を見せる）
- `js/ui.js` — HUDの状態チップ・会話・施策メニュー・エンディング
- `js/main.js` — 起動・チュートリアル・ターン進行の指揮
- 仕様：[docs/SPEC.md](./docs/SPEC.md)
