# 03 — WO_RD → WORLD

**WO_RD → WORLD（ワード・トゥ・ワールド）**

可愛くポップな村を旅する、言葉を覚えないアンドロイド「L」の探索アドベンチャー＋言葉のカードバトル。
「どう勝ったか」が人格（傾向）と結末を決め、進むほど主人公だけが少しずつダークへ反転していく。

> ビルド不要・依存パッケージなし。`index.html` をブラウザで開くだけで動きます（`file://` 可）。
> 外部素材なし（効果音は Web Audio 合成・絵は CSS / SVG のみ）。
>
> **核：見えないゲージを満たすか、見えるゲージを削るか。急ぐ人ほど、無意識に削る側＝凶暴へ流れる。**
>
> 企画の詳細は [`docs/GDD.md`](./docs/GDD.md)。

## 一言コンセプト

ことばで勝つゲーム。勝ち方が、あなたになる。

## このプロト固有のガードレール

このプロトの「絶対に外さない3点」（実装・改善とも、ここを壊す変更はしない）：

1. **二つのゲージを攻めるバトル**
   - **精神HP（見える / mindHp）**：きついことば（harsh）で削る。0で「言い負かした」＝勝利、傾向 **凶暴+**。
   - **思いやりゲージ（見えない / omoiyari）**：やさしいことば（kind）で満たす。満タンで「寄り添った」＝勝利、傾向 **優しさ+**。
   - **思いやりゲージは UI に一切表示しない。** これが心臓。急ぐ人ほど、見えている精神HPを削る側＝凶暴へ無意識に流れる。
   - harsh＝速い（2〜4ターンで勝てる大ダメージ）が凶暴／kind＝遅い（小刻みに積む・5〜8ターン＝敵の攻撃を多く耐える）が優しい。**この非対称が核。**
2. **操作計測（metrics.js）でメタに見透かす**
   - クリック数・テキストのスキップ／読み飛ばし速度・カーソル移動距離を、裏で常時計測。
   - 終盤、「神様的存在（声）」がそれを提示してプレイヤーに語りかける。**「言葉を覚える」ゲームで言葉を飛ばした人ほど、結末で見透かされる。**
3. **word + l = world**
   - タイトルは欠けた綴り「**WO_RD**」。終盤に **L**（主人公の隠し名）を差し込み「**WORLD**」になる気づきを演出。
   - 主人公名 L は隠し設定で、序盤は型番／あだ名（`#L-0`）で呼ぶ。

> 既定テーマは `theme-pop`（可愛くポップ）。凶暴が溜まると body に `dark-1/2/3` が付き、**主人公だけ**の視界が翳り、L のセリフが不穏化する。

## 操作

操作は **マウスのみ**（クリックと、フィールドでのカーソル追従）。キーボードは使いません。

| 操作 | 動作 |
|---|---|
| **クリック「はじめる」** | プロローグ → 村へ。新規ゲーム開始（計測スタート） |
| **クリック「つづきから」** | セーブ（`wo_rd_save_v1`）から再開。**癖（metrics）も復元**されるのでメタ演出が途切れない |
| **フィールドでマウス移動** | 主人公 L が道の上でカーソルに追従して歩く（縦横に自由移動） |
| **ノードに近づく** | NPC＝会話／敵・ボス＝バトル／学校＝ことばショップ／セーブ点＝保存／立て札＝テキスト／出口＝次エリア（必要フラグ達成時のみ） |
| **テキストをクリック** | 1文字ずつ表示中なら全文表示、表示後なら次へ送る（**全文前に送る＝スキップ。裏で計測される**） |
| **バトルでコマンドを選ぶ** | 手札のことばを選ぶ。harsh＝精神HPを削る／kind＝思いやりを満たす（思いやりの数値は出ない） |
| **勝利後の3択** | ことばを1枚ピック（harsh＝凶暴+／kind＝優しさ+）。既知なら Lv 上げ（最大3） |
| **♪ ボタン** | ミュート切り替え |

## 何を試したかったか

- **「見えないゲージ」を勝ち筋の半分にしたら、プレイヤーは無意識にどちらへ倒れるか。** 見える精神HPだけを頼りに削っていくと凶暴へ、見えない思いやりを「効いている感触」だけで満たし続けると優しさへ。UI に出さないことで“急ぐ＝凶暴”の非対称を体感させられるか。
- **裏の操作計測を、終盤に本人へ突き返す体験。** スキップ率・クリック数・カーソル移動距離を貯めておき、神様的存在が「きみは ことばを 急いで 送って いたね」と見透かす。説教ではなく“見えていたよ”という距離感で、テーマ（言葉を覚えるゲームで言葉を飛ばすこと）を刺せるか。
- **タイトルの欠字（WO_RD）を最後まで引っぱり、主人公名 L の差し込みで WORLD にする一発の気づき。** 序盤は型番で呼び、tsukimi の会話などに「ひと文字 欠けて いる」伏線を仕込み、メタ演出で回収する。
- **可愛くポップな村に刃物のことばを混ぜたときの温度差。** harsh は短く尖り（ばか→きえろ→おまえのせい）、kind はやわらかく伸びる（ありがとう→めっちゃ〜）。darkLevel で主人公だけが翳っていく演出が成立するか。
- 参照テンプレートはプロト02（さいごのともだち）。playFx キュー／creatureSVG ピクセルスプライト／オーバーレイ画面／Web Audio 合成／DotGothic16 フォント／render() の switch／トースト の流儀を踏襲。

## ファイル構成

```
03-word-to-world/
├── index.html      画面の骨組み（読み込み順：data→audio→metrics→state→save→battle→cards→field→ui→main）
├── style.css       見た目（ポップな村／不穏化のダーク化テーマ、ピクセルスプライト、フィールドの道）
├── README.md       このファイル
├── docs/
│   └── GDD.md      企画書（正典）
└── js/
    ├── data.js     ★全ゲームデータ（ことば・敵・NPC・エリア・ショップ・結末・メタ・タイトル）＝調整はここだけ
    ├── audio.js    合成SE / BGM（village ポップ・cave 不穏・meta）／darkLevel でデチューン
    ├── metrics.js  ★裏の操作計測（クリック・スキップ・カーソル移動距離）。メタ演出の燃料
    ├── state.js    状態とFSM（傾向 凶暴/優しさ・カードLv・darkLevel・フラグ・結末判定）
    ├── save.js     localStorage 保存／復元（metrics スナップショットも往復＝癖が途切れない）
    ├── battle.js   バトルロジック（二つのゲージ・harsh/kind 処理・敵ターン・勝敗判定）
    ├── cards.js    勝利後の3択カード（harsh/kind 混在・3段階Lv・傾向加点）
    ├── field.js    フィールド（マウス追従の歩行・ノード発火・エリア遷移・結末へ）
    ├── ui.js       描画と入力（各画面 render・typewriter・playFx・creatureSVG・メタ演出）
    └── main.js     起動と配線（タイトル→村→…／クリック計測／rAFループ）
```

> データの正典は `js/data.js`。手触り（敵の硬さ・ことばの威力・暗転しきい値・敵 atk スケール）は原則このファイルの数値だけで変えられます（データ駆動）。
