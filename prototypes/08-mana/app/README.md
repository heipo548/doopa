# 真名 -マナ- 体験版（序章「灯し村」）

> 世界は、名前でできている。

名前を失い消えゆく世界で、漢字の成り立ちそのものを魔法として、万物に名前を取り戻す
記述クラフト・アドベンチャー。SQUARE ENIX GAME CONTEST 2026 提出用デモです。

- 仕様書: [SPEC.md](SPEC.md)（実装の正）
- プレイ時間: 20〜25分

## 起動方法

Node.js 20 以上が必要です。

```bash
npm install
npm run dev        # http://localhost:5173 で起動
```

頒布用ビルド / テスト:

```bash
npm run build      # dist/ に静的ビルド（tsc strict + vite）
npm run preview    # ビルドの確認
npm test           # ユニットテスト（RecipeEngine ほか 27件）
```

※ この Mac では Node を `~/Projects/ai-workspace/.tools/node/bin` に配置しています。
ターミナルで `export PATH=$HOME/Projects/ai-workspace/.tools/node/bin:$PATH` を実行してから上記コマンドを使ってください。

## 操作

| 入力 | 動作 |
|---|---|
| WASD / 矢印キー | 移動 |
| E / Z | 調べる・話す・拾う・鐘を鳴らす |
| Q / X 長押し | きき診（3段階ヒント。困ったらこれ） |
| Space | 名付けモード（「？」マーカーの近くで） |
| Esc | オーバーレイを閉じる |
| マウス | 名付けUI・語彙バッジの操作 |

## あそびかたの芯

1. **拾う** — 世界に散らばる「言ノ葉」（火・日・月・木・点……）を集める
2. **きく** — Q長押しでナナシ（名を失ったもの）の声をきく
3. **名づける** — 単字 / 結字（日＋月＝明）/ 改字（大＋点＝犬）で名前を与える
4. 名を得たものは色を取り戻し、世界が変わる

言ノ葉は消費しません。覚えた字＝あなたの力です。

## デバッグモード

`http://localhost:5173/?debug=1` で起動すると:

- 数字キー 1〜7: 各ゾーンへ瞬間移動
- G: 全語彙を付与
