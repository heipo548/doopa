### 🎬 ことばを獲得する映画 7選

「言葉を覚える＝世界が広がる」を主題に、次の7本を **ストーリー／差別化・革命的ポイント／DOOPA創作への応用** つきで視覚的にまとめた一枚もの。各作品の配色・象徴モチーフから起こした自作 SVG キービジュアル付き（実ポスターではなくイラスト）。

- 奇跡の人 / The Miracle Worker（1962）— 世界の**解像度**が上がる
- ブラック / Black（2005）— 人生の**選択肢**が増える
- ネル / Nell（1994）— 私的言語の双方向の翻訳
- 潜水服は蝶の夢を見る / The Diving Bell and the Butterfly（2007）— 極限の制約からの解放
- メッセージ / Arrival（2016）— 世界の**ルール自体**が変わる
- シェイプ・オブ・ウォーター / The Shape of Water（2017）— 声の外の語彙
- 愛は静けさの中に / Children of a Lesser God（1986）— 並立する言語世界
