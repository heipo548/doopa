# プロト #04 MVP スコープ（MVP_SCOPE）

スラッグ: `04-kotoba-no-android` ／ 仮タイトル: 「なぐれない むら の ルゥ」

MVP の目的：**4つ目の新規プロトとして「最低限遊べる縦スライス1周」を成立させ、デプロイ導線まで通す。**

---

## ✅ 今回入れるもの（IN）

- 既存サイトのプロト一覧に **4番目**として表示（root index.html / 両 README に追記）
- 既存 #03「WO_RD → WORLD」とは **別ページ・別ゲーム**として開ける（`prototypes/04-kotoba-no-android/`）
- **タイトル画面**（仮ロゴ＋プロローグ＋「はじめる」「つづきから」）
- 「はじめる」で開始 → **小さな村フィールド**へ
- **主人公（ルゥ）を操作できる**（クリック/タップ移動＋キーボード代替）
- **NPC に話しかけられる**（こな＝card提供、いどばた ばあ＝フレーバー、最低1体は会話）
- **所持カードが確認できる**（バトルの手札／3択入手画面）
- **言葉カード3択入手**（「どの言葉を知りたい?」→3枚から1枚）
- **口喧嘩カードバトル1戦**（ターン制・敵1体「かどっこ」）
  - **精神HP が見える**（敵の mindHp バー）
  - **思いやりゲージは内部のみ（UI 非表示）**
  - **精神HP を削るルート（harsh）** と **思いやりを満たすルート（kind）の両方で勝てる**
  - 敵ターンでプレイヤー hp が減る（**非パーマデス**）
  - **どちらで勝ったかを記録**（lastWinKind / counters）
- **優しい/凶暴の傾き**を内部状態として記録（tendency.kyoubou / tendency.yasashisa）
- バトル後、**結果/簡易エンド画面**へ（勝ち方×傾向で文面が変化）
- 結果画面で **プレイヤーの選択傾向を短く提示**＋**操作計測の軽いメタ1行**
- **最低限の操作計測**（clicks / skip / dwell / cursor）
- **localStorage 最小セーブ/ロード**（キー `kotoba_save_v1`・metrics スナップショット往復・つづきから）
- **ビルドが通る**（＝静的検証 PASS：JXA ヘッドレス＋構文チェック＋python3 起動）
- **GitHub Pages で公開される構成**（main 直下静的・push で反映）

## 🧩 仮置きするもの（MVP 成立のための仮定・要明記）

> いずれも `HUMAN_DECISION_QUEUE.md` に確認事項として連動。

| 仮置き | 内容 | 確定先 |
|---|---|---|
| 仮タイトル | 「なぐれない むら の ルゥ」（#03 と紛れない別名） | 監督 |
| 主人公呼称 | あだ名「ルゥ」（村人が型番を発音できず付けた、の案）。本名 L は内部キーのみで前面化しない | 監督（Q1/Q2/Q3） |
| 必然性フレーム | 「暴力が禁じられた村＝争いはことばでしか決着できない」 | 監督（Q9） |
| 仮 NPC | こな（パン屋の子・card提供）／いどばた ばあ（フレーバー） | シナリオ |
| 仮 敵 | かどっこ（角っこ）1体・固定行動・atk3 | バトル |
| 仮 カード | harsh 3枚（うるさい/かんけいない/どっか いけ）・kind 4枚（きいてるよ/がんばったね/となりに いるよ/なまえ よんでいい?） | データ |
| 仮 バトル数値 | 敵 mindHp34 / omoiyariNeed36 / atk3、player hp28、winHarsh=winKind=10、useHarsh3>useKind2、darkenAt[8,16,30] | バランス |
| 仮 結果文 | 凶暴寄り/中間/優しい寄りの3トーン | シナリオ |
| 仮 UI/アート | 涼ラベンダー系パレット・CSS/SVG のみ | アート |
| 仮 サウンド | 最小 SE（Web Audio・ON/OFF 可） | サウンド |
| 仮 メタ | 結果画面の軽い1行（skip 傾向で文面変化） | シナリオ |

> **両ルート成立の検算（仮数値）**：harsh＝うるさい(10)×4 で mindHp34→0（4手・hp28→19）／kind＝きいてるよ(6)×6 で omoiyari36 充填（6手・hp28→13、生存）。非対称（4手 vs 6手）成立、kind でも死なない。
