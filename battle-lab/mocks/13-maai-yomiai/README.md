# Battle 13：間合いと言葉の読み合いバトル（The Distance Between Words）

相手との距離感を読み、近づく・離れる・黙るを選ぶ同時選択バトル。

## 遊びの核

相手（裁判所の証人）との **距離（5段階）** を管理しながら、毎ターン **同時選択** で読み合う。
近づきすぎても、遠すぎてもダメ。証人の **癖** を読んで対応する。
「相手に近づけばいいわけではない」「言葉には距離がある」——それを戦闘にする。

## 距離

遠い → 少し遠い → **ふつう** → 近い → 近すぎる（ふつうから始まる）

## 操作（証人と同時に1つ選ぶ）

近づく／離れる／きく／はなす／だまる／てをのばす／なまえをよぶ／うなずく

相性の例：だまる×叫ぶ＝空回り（警戒↓）／てをのばす×近づく＝大成功／近づく×拒絶＝失敗（こころ↓）。
**きく** は読み勝ちでなくても警戒を少し下げる（観察）。

## 証人の癖（乱数なし・読める）

- 近い距離では拒絶しやすい／遠い距離では逃げやすい
- 2回黙った後は叫びやすい／「きく」をされると次に試しやすい

## 終わり方

- **勝利**：ざわめきを0／ふつうを3ターン維持／3回読み勝つ
- **特殊終了**：警戒を0にしてブレイクさせ、*うなずく* か *なまえをよぶ* →「証人は、証言台から一歩だけ降りた。」
- **敗北**：こころが尽きる／時間切れ。
