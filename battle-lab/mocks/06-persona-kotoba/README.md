# 06 ペルソナ参考：心象弱点・総鳴りバトル / Psyche Weakness Battle

Battle Lab のモック。**ペルソナ型の「弱点ターン制バトル」がこの世界観に合うか**を触って試すための検証用モックです（完成版ゲームではありません）。

公開URL（GitHub Pages）:
**https://heipo548.github.io/doopa/battle-lab/mocks/06-persona-kotoba/**

## 確認したい核（このモックの目的）

```
ほころび(弱点)を突く → 心崩れ → もう一言 → 対話 or ことばの総鳴り
```

この一連の流れが「テンポよく・気持ちいいか」を判断するためのモックです。

## あそびかた

1. 相手（村長の影）は**仮面**をかぶっている。仮面ごとに**ほころび（弱点）／こわばり（耐性）**が違う。
2. 弱点・耐性は最初 `？？？`。**言葉技を一度使うと**、そのタイプの結果が開示される。
3. **ほころび**を突くと相手が**心崩れ**になり、Lに**もう一言**（追加1回）が発生する。
4. 心崩れ中は **対話（たずねる）** か **ことばの総鳴り**（閉じた心 -18）が選べる。
5. 対話で仮面が変化し、やがて**本音**にたどり着く。本音で「言葉を教えて」を選ぶと**対話勝利**。
6. 閉じた心を0にしても勝ち。Lの**ノイズ耐性**が0になると敗北。

## 仮面と弱点（ネタバレ）

| 仮面 | ほころび（弱点） | こわばり（耐性） |
|---|---|---|
| 疑い（初期） | 問い・沈黙 | 感謝・約束 |
| 不安 | 安心・沈黙・約束 | 拒絶・問い |
| 沈黙 | 沈黙・名前 | 感謝・あいさつ |
| 隠し事 | 問い・名前・沈黙 | あいさつ・感謝 |
| 本音 | （なし） | （なし）※対話で勝利 |

仮面が変わる主なきっかけ：`どうして？/なまえ`の追加効果(30%)→隠し事 ／ 心崩れ中の「こわいの？」→隠し事 ／ 沈黙する2回→沈黙 ／ 村長の「心配する」→不安 ／ 閉じた心≤10→本音。

## ファイル構成（他モックと同じ4分割）

| ファイル | 役割 |
|---|---|
| `index.html` | 画面の骨組み |
| `style.css` | 見た目（仮面・？？？・心崩れ・総鳴りの雰囲気） |
| `js/data.js` | 仮面・言葉技・村長の行動・対話・台詞（**データだけ**。ここを足せば広がる） |
| `js/engine.js` | 進行ルール（**DOM非依存の純粋ロジック**） |
| `js/ui.js` | 描画（DOMはここだけ） |
| `js/main.js` | 配線（クリック→engine→再描画） |
| `tools/headless-check.js` | DOM非依存コアの自動検証（JXA） |

## ローカルで動かす

```sh
# そのまま開くだけでも動く
open index.html

# ロジックだけの自動検証（ブラウザ不要・JXA）
MOCKDIR="$PWD" osascript -l JavaScript tools/headless-check.js
```
