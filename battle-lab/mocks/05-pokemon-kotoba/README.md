# 05 — Pokémon参考：言葉タイプ相性バトル / Word Type Battle

Battle Lab のバトルモック。**ポケモン型のターン制バトル**が、このゲームの世界観に合うかを試すための検証用モックです（完成版ではありません）。

公開URL（GitHub Pages）:
**https://heipo548.github.io/doopa/battle-lab/mocks/05-pokemon-kotoba/**

ローカルでは `index.html` を開くだけで動きます（依存ゼロ）。

## このモックで確認したいこと

> **「相手の感情タイプを見て、どの言葉を使うか選ぶのが楽しいか」**

敵を倒すのではなく、村長の「閉じた心」を言葉でほどくのが目的です。

## 遊び方

1. 村長の **感情タイプ**（疑い／不安／沈黙／隠し事）を見る。
2. Lが覚えている **4つの言葉技** から1つを選ぶ。
3. 言葉タイプ × 感情タイプの **相性** で「届いた量」が変わる。
   - よく届く＝1.5倍 / 普通＝1.0倍 / 少しずれる＝0.5倍
4. 村長も毎ターン1つ行動する。**受容**でノイズを防ぎ、**残響**（使用回数）を切らさないように。
5. 村長の「閉じた心」が0で **勝利**、Lの「ノイズ耐性」が0で **敗北**。

## 用語の置き換え（ポケモン用語 → この世界）

| ポケモン | このゲーム |
|---|---|
| HP（自分） | ノイズ耐性 |
| HP（相手） | 閉じた心 |
| PP | 残響 |
| ダメージ | 届いた量 / ほどいた量 |
| 状態異常 | 心の乱れ（迷い・警戒・ほころび…） |
| 効果抜群 | 言葉が、よく届いた |
| いまひとつ | 言葉は、少しだけずれた |
| ポケモンセンター | 学校 / 保健室（将来） |

## 実装メモ（構成）

- `index.html` … 画面の骨組み（4技選択UI・感情タイプ表示・ログ・勝敗オーバーレイ）
- `style.css` … 見た目（白〜淡色ベース、ポップだが少し不穏。言葉タイプごとの色分け）
- `js/data.js` … **中核データ**：言葉タイプ8種・感情タイプ・相性表・言葉技・村長・状態異常・世界観テキスト
- `js/engine.js` … **DOM非依存の進行ルール**：相性／命中／効果量／状態異常／村長の行動／勝敗
- `js/ui.js` … 描画（`state` を読んでHTMLに反映するだけ）
- `js/main.js` … 配線（クリック→engine→再描画。L→村長を交互に処理）
- `tools/headless-check.js` … JXAでコアを回帰テスト（下記）

### 自動チェック（ブラウザ不要・JXA）

```sh
MOCKDIR="$PWD" osascript -l JavaScript tools/headless-check.js
```

初期値・相性・効果量（こんにちは 8×1.5=12／警戒-20%／迷い-30%／ほころび+50%）・受容軽減・
命中失敗・感情変化・勝敗メッセージ・1戦まるごとのオートプレイ（例外なし・必ず決着）を確認します。

## 実装済み / 未実装

**実装済み（必須＋一部の任意）**
- Lの4技（こんにちは／だいじょうぶ／どうして？／沈黙する）＋タイプ・効果量・命中・残響
- タイプ相性（4感情ぶんの相性表）と相性ログ
- 村長の感情タイプと **動的な変化**（隠し事化／沈黙化／不安化）
- 状態異常：迷い・警戒・ほころび・ぶれ・しずけさ／受容
- 村長の行動4種（疑う・問いかける・黙って見る・心配する）
- 勝敗・勝利/敗北メッセージ・リスタート・Battle Labへの導線
- 勝利後の **報酬選択UI（表示のみ）** ＝ ありがとう／まって／なまえ

**未実装（将来の器だけ用意）**
- 言葉の習得・入れ替え（同時に4つまで）：`STARTING_MOVES` に id を足すだけで装備できる構造
- 言葉のレベルアップ（3段階）：各 `MOVE_LIBRARY[*].levels` を埋める想定
- 学校（ショップ兼回復施設）：残響回復・状態異常治療・相性を教わる など
- 将来の感情タイプ（怒り・悲しみ・罪悪感・期待・本音）も `EMOTION_TYPES` に器だけ定義済み

## 新しい言葉技や相手を足すには

`js/data.js` の `MOVE_LIBRARY` / `ENEMIES` / `MATCHUP` を編集するだけ（`engine.js` は触らない）。
追加技は `effect` キー（`damage` / `selfJuyou` / `bonusVs` / `changeEmotion` …）の組み合わせで表現できます。
