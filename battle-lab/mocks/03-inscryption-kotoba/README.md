# 03 — Inscryption参考「言葉を捧げる盤面バトル」

英語表記：**Word Offering Board Battle**

Battle Lab（バトルシステム試作場）の3つめのモック。
**盤面（声の道）に言葉を置き、強い言葉を出すために 別の言葉を供えながら、
村長の「閉じた心」に届かせる** 盤面バトルの検証用。

> これは完成版ではなく、バトルシステム検証用のモックです。
> 試したい一番の感覚は——「言葉を覚えるゲームなのに、強い言葉を出すために、
> 別の言葉を手放さなければならない」という、少し気持ち悪い不穏さです。

## 遊び方

1. 毎ターン、山札から 言葉を 1枚 引く（初手は4枚）。
2. **手元の言葉をクリック** → 必要なら **場の自分の言霊をクリックして供える** → **空いた声の道をクリックして置く**。
3. 「ターン終了」を押すと、声の道ごとに 言葉が動く。
   - 正面に村長の言葉があれば **ぶつかる**（互いの届く力ぶん こわれにくさが減る）。
   - 正面が空なら、村長の **閉じた心** に届く。
4. 村長の **予告** が翌ターン 盤面に出てくる。
5. 村長の **閉じた心**（10）を 0 にすれば勝ち。L の **ノイズ耐性**（10）が 0 になると負け。

## 初期語彙束（言葉札）

| 言葉 | 供え | 届く力 | こわれにくさ | 性質 | はたらき |
|---|---|---|---|---|---|
| こんにちは ×4 | 0 | 1 | 2 | — | はじめて覚えた言葉 |
| だいじょうぶ ×3 | 0 | 0 | 3 | 守る | ぶつかりで受けるダメージ -1 |
| どうして？ ×2 | 1 | 1 | 1 | のぞく | 出たとき さらに先の予告を のぞく |
| 沈黙する ×1 | 0 | 0 | 4 | 壁 | 相手を止めるが 閉じた心に届かない |
| 信じて ×1 | 2 | 3 | 2 | まっすぐ | 正面が空のとき 強く届く（+1） |
| もう一度 ×1 | 残響2 | — | — | （残響カード） | 捨て札からランダム1枚を手元へ |

## 言葉の性質

- **守る** … 正面の相手から受けるダメージ（こわれにくさ）を1減らす。
- **のぞく** … 場に出たとき、村長の さらに先の予告（気配）を1枚 のぞき見る。
- **壁** … 相手の言葉を止めるが、閉じた心／ノイズ耐性には届かない。
- **まっすぐ** … 正面が空いているとき 強く届く（届く力 +1）。
- **重い**（村長の「問い」）… 場にある間、こちらの供え言葉コストが +1。
- **ほどける**（村長の「心配」）… 壊さず 数ターン残すと、村長の閉じた心が少し ほどける。

## 相手：村長（閉じた心 10）

悪者ではない。主人公を疑い、怖がり、でもどこか心配している。予告として1ターン前に見える。

|---|---|---|---|
| 疑い | 1 | 2 | — |
| 問い | 0 | 3 | 重い（供えコスト +1） |
| 沈黙 | 0 | 4 | 壁 |

## 不穏ギミック（少しだけ「ルールがこちらを見ている」感じ）

1. **喋る言葉札** … 数ターンに一度、手元や場の言葉が短いひとことを漏らす（沈黙する「……」など）。
2. **供えた言葉を覚えている** … 同じ言葉を2回以上 供えると、盤が それを覚えている短い文を出す。
3. **名前のグリッチ** … ノイズ耐性が低くなると、言葉の名前に淡いグリッチがかかる（視覚のみ・可読性は保つ）。

## サブ資源：残響

言葉が **壊れる / 供えられる** たびに 残響が +1。
今回はそれを使う最初の一枚 **「もう一度」**（残響2で 捨て札から1枚回収）を入れてある。
将来は「言葉を失ったあとにだけ使える言葉」を増やしていく器。
