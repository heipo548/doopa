# Battle 09：ことばアクションコマンドバトル（How You Say It）

同じ言葉でも、届けるタイミングによって効果が変わるコマンドバトル。

## 遊びの核

コマンドを選んだあとに **タイミング入力** がある。中央で「いま！」を押すほど深く届く。
早すぎると **急かし**、遅すぎると **届かない**。そして **だまる**＝何もしないのが正解の場面もある。
これは音ゲーではなく「言葉を届ける手触り」を試すもの。

## 操作

1. コマンドを選ぶ。
2. **タイミング系**（きく／はなす／ふれる／ありがとう／本当にありがとう）はトラックが流れる。中央の緑帯で「いま！」。
   - 判定：ふかく届いた／届いた／少し届いた／届かなかった／急かしてしまった
3. **だまる** は待つのが正解。バーが満ちれば成功、途中で話すと急かし。
4. **めっちゃありがとう**（Lv3）はポップだが、この先生には軽すぎて逆効果。

警戒が高いほど、ざわめきを下げる効果が鈍る。きく／だまるで警戒を下げる。

## 終わり方

- **勝利（ざわめき）**：ざわめきを 0 に。
- **勝利（連続）**：正しく 3 連続で届ける。
- **特殊終了**：警戒を 0 にしてさらに *だまる* → 授業が止まる →「黒板の world から、l だけが消えた。」
- **敗北**：こころが尽きる。

## 構成（依存ゼロ・静的）

`index.html` / `js/data.js`（数値・文章）/ `js/engine.js`（判定 `resolveCommand(id,pos)`・`chooseSilence(waited)`）/
`js/ui.js`（描画）/ `js/main.js`（タイミングのアニメ＋配線）/ `tools/headless-check.js` / `tools/dom-smoke.js`
