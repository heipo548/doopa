# Battle 14：旅支度オートバトル（What You Packed）

カバンに詰めた言葉や思い出が、半自動で戦い方を決める準備型バトル。

## 遊びの核

戦闘前に **カバン(3×3)** へ持ち物を配置する。**配置とシナジー** で戦い方が変わり、
戦闘は **半自動** で進む。プレイヤーは見守るか、ひとつ使うかで介入する。
「どんな気持ちで旅に出たかが、戦い方になる」——**準備が、すでに戦い**。

## 持ち物（9種）

手紙／写真／ありがとう／だいじょうぶ／石／枝／小さな鈴／パン／**空白**
（言葉＝赤系・物＝緑系・空白＝紫系）

## シナジー（隣どうし）

- 手紙＋写真 → 敵の記憶を開示（写真が強くなる）
- ありがとう＋手紙 → 最後に特別な言葉が届く（手紙が強くなる）
- パン＋だいじょうぶ → 回復アップ
- 石＋枝 → 防御アップ
- 鈴＋空白 → 敵の初回行動を遅らせる

## 操作

1. **準備**：持ち物を選び、カバンのマスをタップして配置。シナジーはその場で確認できる。
2. **「この支度で旅に出る」** で戦闘開始（半自動）。
3. **見守る**（1つ進める）／**ひとつ使う**（カバンから選んで先に発動）。手紙は最後に発動する。

## 結末（どう詰めたか で分岐）

- **説得（persuade）**：言葉中心で ざわめき0 → 言葉が返ってくる。
- **懲らしめ（punish）**：物中心で ざわめき0 → 逃げるが言葉は戻る。
- **見逃す（letgo）**：削りきれず → 一部の言葉は戻らないが、別の記憶を得る。
- **空白エンド（blank）**：**空白を中央(真ん中)に置く** と起きる特殊終了 →「カバンの真ん中には、何もなかった。だから、悪党はそこに言葉を置いた。」
