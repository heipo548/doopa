# Battle 08：こころの時間差バトル（Timeline of the Heart）

敵の感情が爆発する前に、言葉を間に合わせるタイムライン式バトル。

## 遊びの核

ターン制ではなく、**敵味方の行動が一本の時間軸（tick）に並ぶ**。
自分の言葉には「発動までの待ち時間（cast）」があり、村長の**爆発ゲージ**が満ちる前に
強い言葉を *間に合わせられるか* を考える。「待つ」「軽い言葉で早く動く」「重い言葉で大きく変える」
の選択そのものが戦略になる。

## 操作

- **きく**（cast1）：早い。村長の次の予兆を読み、爆発ゲージを少し下げる。
- **こえをかける**（cast1）：早い。爆発ゲージを下げる。
- **なまえをよぶ**（cast3）：遅い。ざわめきを下げる。秘密を言いかけた村長を止められる。
- **だまる**（cast1・コスト0）：おちつきを取り戻し、近すぎる親切に強くなる。
- **てをのばす**（cast3）：遅い。ざわめきを大きく下げる。が、爆発に間に合わないと無駄になる。

`cast` ぶん時間が進む間に村長が動く。タイムラインで「村長の次の行動／あと何 tick」を読む。

## 終わり方

- **勝利（ざわめき）**：村長のざわめきを 0 にする。
- **勝利（耐久）**：爆発させずに一定ターン耐える。
- **特殊終了**：村長が「村の秘密を言いかける」のを *なまえをよぶ* で止め、続けて *きく* に成功 → 「村長は、笑うのをやめた。」
- **敗北**：爆発ゲージが満ちる／こころが尽きる。

## 構成（依存ゼロ・静的）

- `index.html` … 画面の骨組み（状態・タイムライン・コマンド・ログ・勝敗オーバーレイ）
- `js/data.js` … 数値・村長の行動表・コマンド・文章
- `js/engine.js` … 進行ルール（DOM非依存・`state` を関数で書き換える）
- `js/ui.js` … 描画のみ（`UI.render()`）
- `js/main.js` … engine と ui の配線（`Game.*`）+ 起動
- `tools/headless-check.js` … engine の自動点検（JXA）
- `tools/dom-smoke.js` … UI配線の煙テスト（JXA）

## テスト

```sh
MOCKDIR="$PWD" osascript -l JavaScript tools/headless-check.js   # ルール 19件
MOCKDIR="$PWD" osascript -l JavaScript tools/dom-smoke.js        # UI配線 6件
```
